swift 闭包

发表于 2016-09-27

闭包样式

{(参数1,参数2) -> 返回值
    in
}

解释

闭包的函数整体部分由关键字 in 导入,这个关键字表示闭包的形式参数类型和返回类型定义已经完成,并且闭包的函数体即将开始.

函数和闭包都是引用类型

逃逸闭包

当闭包作为一个实际参数传递给一个函数的时候,我们就说这个闭包逃逸了.

C语言的指针 引用和取值

发表于 2016-09-23

基本概念

指针

指针是一个存储计算机内存地址的变量.

取值

从指针指向的内存读取数据称作指针的取值.指针可以指向某些具体类型的变量地址,例如int,long…

*

操作符 :
既可以声明一个指针变量,也可以用作指针的取值.
当生命一个变量的时候, 用来表示声明了一个指针.其他情况表示指针的取值.

&

地址操作符: & :
用来引用一个内存地址.通过在变量名字前使用 & 操作符,我们可以得到该变量的内存地址.

// 声明一个int指针
int *ptr;
// 声明一个int值
int val = 1;
// 为指针分配一个int值的引用
ptr = &val;
// 对指针进行取值，打印存储在指针地址中的内容
int deref = *ptr;
printf("%d\n", deref);
